PROJETOS

Projeto Cordel

Este projeto foi desenvolvido com foco em estrutura semântica e estilização avançada com CSS. A proposta é apresentar um conteúdo em formato de cordel com uma experiência visual imersiva, utilizando efeitos de paralaxe, imagens de fundo fixas e tipografia expressiva. O objetivo principal foi treinar organização de layout, controle de espaçamento, responsividade e aplicação de estilos que valorizam a leitura e a estética do conteúdo.

Projeto Android

Neste projeto foi criada uma página completa sobre a história do Android, com estrutura organizada e design moderno. O desenvolvimento priorizou a construção de um layout bem distribuído, uso correto de seções semânticas, incorporação de mídias como vídeos e imagens, além de estilização consistente para melhorar a experiência do usuário. O foco foi praticar hierarquia visual, organização de conteúdo informativo e responsividade para diferentes tamanhos de tela.

Projeto Redes

Este projeto simula uma interface interativa de navegação entre redes sociais, com foco em posicionamento de elementos, interatividade visual e adaptação para dispositivos móveis. Foram aplicados conceitos de layout flexível, responsividade e controle de elementos visuais em diferentes resoluções. O principal objetivo foi desenvolver habilidades de estruturação visual, alinhamento e comportamento responsivo da interface.

Quem sou eu?
